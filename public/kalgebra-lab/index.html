Runs the exact examples.ipynb from the public KAlgebra package, executed cell-by-cell in a Pyodide kernel on a Web Worker (so the page stays responsive). When the kernel is ready, press Run all cells or any cell's Run button; edit a cell and re-run it too. First load fetches Pyodide (~10 MB) and the library once, then it is browser-cached. Nothing runs until you click.
